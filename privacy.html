Privacy Policy / 隐私政策

Effective and last updated: July 13, 2026 / 生效及最后更新日期：2026年7月13日

1. Scope / 适用范围

This policy explains how this static artist portfolio website handles information. The website presents artworks, biographical information, professional records, publications, and contact details for Cher Wang / 王凯萱.

本政策说明本静态艺术家作品网站如何处理信息。网站用于展示王凯萱的作品、艺术家简介、专业履历、出版物及联系方式。

2. Information collected by this website / 本网站收集的信息

This website currently has no user accounts, comments, contact forms, checkout, payment processing, or file-upload functions. The website owner does not currently operate visitor analytics and does not intentionally collect personal information through the website itself.

本网站目前不提供用户账户、评论、联系表单、结账、支付处理或文件上传功能。网站所有者当前未启用访问统计，也不会通过网站本身主动收集个人信息。

If analytics are enabled in the future, this policy will be reviewed and updated as appropriate before or when that change takes effect.

如未来启用访问统计，本政策将在该功能启用前或启用时按实际情况进行审查和更新。

3. GitHub Pages hosting / GitHub Pages 托管

The website is hosted by GitHub Pages. When you request a page or asset, GitHub may process technical and usage information needed to deliver and operate its service, such as IP address, device or browser information, request time, and requested resource. That processing is controlled by GitHub under its own policies, not by this website.

本网站由 GitHub Pages 托管。当您请求页面或资源时，GitHub 可能为提供和运行其服务而处理必要的技术及使用信息，例如 IP 地址、设备或浏览器信息、请求时间和所请求的资源。该处理由 GitHub 按其自身政策负责，并非由本网站控制。

For details, see the GitHub General Privacy Statement.

4. Cookies and analytics / Cookie 与访问统计

This website does not currently set its own cookies or use local storage for tracking. Google Analytics 4 and Plausible are not currently enabled, and no tracking identifier has been configured. GitHub may use cookies or similar technologies in connection with its own services under GitHub's policies.

本网站当前不会自行设置 Cookie，也不会使用本地存储进行跟踪。Google Analytics 4 与 Plausible 当前均未启用，且未配置任何追踪标识。GitHub 可能依据其自身政策，在提供服务时使用 Cookie 或类似技术。

5. Email and external services / 电子邮件与外部服务

If you contact the artist by email or through a linked third-party platform, the information you choose to send is handled through that service and may be retained as reasonably needed to respond or maintain correspondence. Please do not send passwords, payment-card information, government identifiers, or other unnecessary sensitive information.

如您通过电子邮件或所链接的第三方平台联系艺术家，您主动发送的信息将通过相应服务处理，并可能在回复或保留往来记录所合理需要的期限内保存。请勿发送密码、支付卡信息、政府签发的身份标识或其他不必要的敏感信息。

6. Data requests and changes / 信息请求与政策变更

Questions or requests concerning correspondence sent directly to the artist may be submitted through the Contact page. This policy may be updated when the website's actual features or legal obligations change. The date above identifies the current version.

如对直接发送给艺术家的往来信息有疑问或请求，可通过联系页面提出。当网站实际功能或适用义务发生变化时，本政策可能更新。页面顶部日期标识当前版本。
